**Paste a screenshot. Style it. Copy it back.** That's it.

## Features

📋 **Paste, drop, or upload** — `Ctrl+V` works anywhere, no click needed

🎨 **Gradient, solid, or transparent backgrounds** — with angle control and one-click randomize

🖼️ **Browser frames** — macOS and Windows chrome with editable URL bar

🔄 **3D transforms** — tilt, rotate, and perspective for depth

🌗 **Shadows, corners, padding, scale** — every control has a slider, text input, and per-field reset

🎲 **Presets & randomize** — Default, Mockup, Minimal, or roll the dice

🔍 **Canvas zoom** — slider, `Ctrl+scroll`, or pinch-to-zoom anchored to your cursor

📱 **Mobile-ready** — responsive layout with bottom-sheet settings and touch zoom

💾 **Copy or download** — 2x retina PNG, right-click to copy for the fastest workflow

🌙 **System theme** — follows your OS light/dark preference

⚡ **Settings persist** across sessions — auto-expanding sections, smart overflow padding

## Setup

```bash
bun install
bun dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `bun dev` | Start dev server |
| `bun run build` | Type-check + production build |
| `bun run test` | Run tests with Vitest |
| `bun run lint` | Lint with Biome |
| `bun run format` | Format with Biome |

## Stack

React 19 · Vite 7 · TypeScript · Tailwind v4 · shadcn/ui · Zustand · html-to-image · Vitest · Biome · Bun
